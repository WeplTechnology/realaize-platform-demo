## Overview

This is an interactive prototype showcasing the complete architecture of the Realaize Capital platform:

- **Dashboard** — Real-time operational overview with KPIs, activity feed, and alerts
- **Pipeline** — 6 Kanban sub-pipelines (Assets, Proposals ×4, Needs, Investors, Operators, Lenders)
- **CRM** — 5 directories with detailed entity cards
- **Workspace** — Personal task management, email integration, calendar sync
- **Notifications** — Complete alert system
